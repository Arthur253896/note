# 时间序列预测

## 研究现状
评估从过去的一系列数据得出的未来预测称为时间序列预测。用于时间序列预测的三种技术是随机模型、人工神经网络和支持向量机。ARIMA 是广泛用作随机模型的著名技术，在最大研究领域中用作基础版本 [ 10 , 11]。这里的数据是线性的，并且服从正态分布。随机模型对线性数据执行详尽，而实际时间序列数据被认为是非线性的。具有独特功能的实时数据中的随机模型中的约束，例如缺少值，多维和非线性数据的相关性不大。由于它们的理想特性，神经网络是比随机模型更合适的技术。神经网络专业知识，可以在没有专业知识和数据推测的情况下掌握大量线性和非线性信息 [ 12]。到目前为止，已经建立了不同种类的神经网络，其中一些已被用于预测时间序列。例如，ANN 使用多层感知器来预测时间序列数据。时滞神经网络 [ 13] 用于计算时间序列的隐藏特征，用于预测航空公司乘客的时间序列数据。MLP 的一些出色功能使它们能够很好地用于时间序列计算，包括非线性建模、抗噪性、提供预测度量的输入和输出数量的柔韧性以及多维功能的改进。另一种类型的 NN 是卷积神经网络 (CNN)。CNN 过滤器检查周期性趋势并预测时间序列数据。一维 CNN 通常用于预测时间序列数据。确定的值作为 NN 的输入，多层 NN 的未确定值被预测为输出。WaveNet 卷积神经网络是另一种时间序列预测模型，其中序列特征与值不相关。这里没有使用池化层，因此 NN 的输入和输出的规模是相等的。这种技术太快了，因为没有使用循环网络并且层和内核的大小增加了。长短期记忆 (LSTM) 是循环神经网络 (RNN) 的另一个组件，用于预测时间序列数据 [14、15 ]。_ 以前的记录通过使用循环网络来使用。LSTM 通过同时访问先前状态输入数据和当前状态输入数据来预测未来数据，从而提供正确的评估。使用的另一种 LSTM 形式是 Stacked-LSTM，它利用 LSTM 堆栈来评估复杂的模式。双向 [ 16] 是 LSTM 的第二个版本，它同时使用前向和后向层进行进展。在时间序列预测中使用 CNN 和 LSTM 的组合是不同的。CNN-LSTM 是空间和时间模型的组合。一维 CNN 访问空间数据作为输入，并伴随着带有时间数据的 RNN。神经网络社区的另一种技术是编码器和解码器模型。编码器对位于选定模式上的输入数据进行编码，然后解码器根据编码数据对输出数据进行解码，以提供可靠的输出。该模型具有更大的上下文视角，可提供良好的性能。将这类网络的广泛类型引入时间序列预测。即，具有注意力的 LSTM 序列到序列模型 [ 17 - 19]。在这个模型中，编码输入对输出的影响是根据不同的专业问题依次管理和实施的。由于不同时间序列中的数据非线性，在长期依赖性方面存在挑战。为了克服上述问题，提出了 TTPM 用于计算长期序列，以提高模型的整体性能
